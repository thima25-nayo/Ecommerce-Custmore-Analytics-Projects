## 🎯 Business Problem
The executive leadership team lacked automated visibility into international transactional revenue trends, regional client purchasing concentrations, and shifting order velocities. This data gap caused manual reporting lag and limited the team's ability to spot customer churn patterns early.

## 🛠️ Data Architecture & Pipeline
- **Data Layer:** Extracted and profiled over 500,000 global transaction records via an open-source retail framework.
- **ETL Transformation (Power Query):** Executed string scrubbing to eliminate null customer segments, isolated product returns by filtering unique alphanumeric identifiers, and enforced correct decimal numerical values for unit margins.
- **Data Modelling:** Modelled specialised DAX measures establishing clean aggregate values for total transactional revenue, unique operational client footprints, and average order value (AOV) tracking metrics.

## 📈 Key Insights & Recommendations Delivered
1. **Revenue Concentration:** Discovered that a single regional market generated over 40% of the aggregate sales pipeline, prompting a recommendation to diversify geographic marketing campaigns.
2. **Operational Efficiency:** Identified a consistent 14% drop-off in processing volume during seasonal Q3 intervals, highlighting an optimal window for system maintenance and inventory adjustments.
3. **Reporting Optimisation:** Automated an interactive regional slicing interface that completely replaced manual weekly dashboard distributions, saving 5 hours of administrative overhead per week.
